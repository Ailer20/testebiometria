Ajuda e FAQ

Primeiros Passos

Como faço login no sistema?

Para acessar o sistema:

  1. Acesse a página inicial
  2. Digite seu email institucional
  3. Digite sua senha
  4. Clique em "Entrar"
Como alterar minha senha?

Para alterar sua senha:

  1. Acesse "Configurações" no menu lateral
  2. Clique em "Alterar Senha"
  3. Digite sua senha atual
  4. Digite e confirme sua nova senha
  5. Clique em "Salvar"
Como configurar meu perfil?

Para configurar seu perfil:

  1. Acesse "Configurações"
  2. Atualize suas informações pessoais
  3. Adicione uma foto de perfil
  4. Configure suas preferências de notificação

Registro de Frequência

Como iniciar o registro de frequência?

Para registrar a frequência:

  1. Acesse a seção "Registro de Frequência"
  2. Selecione a turma
  3. Clique em "Iniciar Monitoramento"
  4. Escolha entre reconhecimento facial ou biometria
Como cadastrar a biometria dos alunos?

Para cadastrar biometria:

  1. Acesse "Configurações"
  2. Selecione "Gerenciar Biometrias"
  3. Escolha a turma e o aluno
  4. Siga as instruções na tela para cadastrar a digital

Observação: É necessário que cada aluno esteja presente para o cadastro.

Como cadastrar foto para reconhecimento facial?

Para cadastrar reconhecimento facial:

  1. Acesse "Configurações"
  2. Selecione "Gerenciar Reconhecimento Facial"
  3. Escolha o aluno
  4. Capture fotos do rosto em diferentes ângulos
  5. Confirme o cadastro

Relatórios

Como exportar relatórios de frequência?

Para exportar relatórios:

  1. Acesse a seção "Relatórios"
  2. Selecione o período desejado
  3. Escolha a matéria
  4. Clique em "Exportar PDF" ou "Exportar CSV"
Como analisar as estatísticas de frequência?

No painel de estatísticas você encontrará:

  • Gráficos de frequência por período
  • Taxa de presença por aluno
  • Comparativo entre turmas
  • Tendências de frequência

Dica: Utilize os filtros disponíveis para análises específicas.

Solução de Problemas

O que fazer se o reconhecimento facial não funcionar?

Se o reconhecimento facial não estiver funcionando:

  • Verifique a iluminação do ambiente
  • Certifique-se que a câmera está funcionando
  • Tente utilizar a biometria como alternativa
  • Entre em contato com o suporte técnico
Problemas com o leitor biométrico?

Soluções comuns:

  • Limpe o sensor do leitor
  • Verifique a conexão USB
  • Reinicie o computador
  • Recadastre a digital se necessário
Como resolver problemas de exportação de relatórios?

Se houver problemas ao exportar relatórios:

  • Verifique sua conexão com a internet
  • Limpe o cache do navegador
  • Tente outro formato de exportação
  • Verifique se há atualizações pendentes

Segurança e Privacidade

Como os dados são protegidos?

Nosso sistema utiliza:

  • Criptografia de ponta a ponta
  • Armazenamento seguro de dados biométricos
  • Backup automático
  • Conformidade com LGPD
Políticas de privacidade dos dados

Nossos compromissos:

  • Dados usados apenas para controle de frequência
  • Acesso restrito a pessoas autorizadas
  • Não compartilhamento com terceiros
  • Exclusão segura quando solicitado
